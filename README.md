# WeMarine Website

Sito web vetrina statico del progetto WeMarine.

## Obiettivo
Presentare servizi e informazioni del brand con un sito leggero, veloce e facilmente manutenibile.

## Struttura progetto
- `index.html` -> Home page
- `pages/about.html` -> Pagina About
- `pages/service.html` -> Pagina Servizi
- `assets/css/main.css` -> Stili globali
- `assets/js/main.js` -> Script globali
- `assets/img/` -> Immagini e media

## Avvio in locale
1. Aprire la cartella progetto in VS Code.
2. Avviare con estensione Live Server (consigliato), oppure aprire `index.html` nel browser.

## Workflow Git
- `main` -> versione stabile/pubblicabile
- `dev` -> sviluppo continuo
- `feature/*` -> nuove funzionalità (es: `feature/lang-switch`)

## Convenzioni
- Commit piccoli e descrittivi.
- Una feature per branch.
- Merge su `dev`, poi su `main` quando stabile.
